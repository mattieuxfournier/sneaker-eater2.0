Easy Mode

Although this 2D shooter game was inteaded to be A SHOOTER GAME, you can just admire the art work of the game by simply not touching the gun. This turns the game from a violent shooter to a peaceful side scroller!

Default Mode

The Default Mode is how the game is inteaded to be played. Start your journey by picking up the gun and shooting all the cops in order to escape the mall, this mode has challenging parts, you can only take one hit and some difficult puzzles. Test your luck and see if you can beat Sneaker Eater.

Cheater Mode

Cheater Mode is the third and final way to play this game, in cheater mode you can find secret rooms and basically skip the entire game, this makes this mode ideal for people who are looking for a game to finish as fast as they can. Although these are 3 different modes you cannot select them in the starting screen, the way you discover which mode your playing is based off of how you choose to play our game.
